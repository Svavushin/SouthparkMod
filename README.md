# 🎮 SouthparkMod

> South Park themed content for RimWorld  
> Тематический мод по South Park для RimWorld

**Supports / Поддержка:** RimWorld 1.6  
**Requires / Требуется:** Harmony

---

## 🇬🇧 English

## ✨ Features

### 💀 Kenny Death
- Plays Kenny death sound
- Dresses the corpse in Kenny outfit

### ✝️ Resurrection Popup
- Shows Jesus image when using resurrector mech serum

### 🦹 Mysterion
Replaces Stranger in Black with Mysterion:

- Melee 20
- Nimble trait
- Frag grenades
- x2 body part HP
- Auto-revive after ~2 seconds
- Up to 4 resurrections

### 💩 Mr. Hankey Event
Once per year:

- Arrives from the map edge
- Walks to the colony
- Drops multiple random gifts
- Can give animals
- Wanders near colonists
- Leaves the map

## 🇷🇺 Русский

## ✨ Возможности

### 💀 Смерть Кенни
- Проигрывается звук смерти Кенни
- Труп переодевается в костюм Кенни

### ✝️ Попап воскрешения
- При использовании resurrector mech serum появляется изображение Иисуса

### 🦹 Мистерио
Заменяет Stranger in Black на Мистерио:

- Melee 20
- Черта Nimble
- Гранаты
- x2 здоровье частей тела
- Авто-воскрешение через ~2 секунды
- До 4 воскрешений

### 💩 Событие Мистера Какашки
Раз в год:

- Появляется с края карты
- Приходит к колонии
- Несколько раз оставляет случайные подарки
- Может подарить животных
- Гуляет рядом с колонистами
- Уходит с карты

## 📥 Installation / Установка

1. Download the mod / Скачайте мод
2. Place it into `RimWorld/Mods` / Поместите в `RimWorld/Mods`
3. Enable it in the mod list / Включите в списке модов
